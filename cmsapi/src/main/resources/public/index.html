Get Started

Login into your existing account or register a new account.

Type Endpoint Value Params
POST /subscription/create Create a new account subscription as community owner. If your email is already registered as a community owner, overwrite current subscription with new one. Subscription can only be overwritten by community owner. String subscriptionPlan, String email
POST /auth Login into account. Api with authenicate and authorize account and designate permissions based off of account role. username, password

Subscription

All endpoints related to subscription. Can only be used by community Admin.

Type Endpoint Value Params
POST /subscription/create Create a new account subscription as community owner. If your email is already registered as a community owner, overwrite current subscription with new one. Subscription can only be overwritten by community owner. String subscriptionPlan, String email
POST /subscription/cancel Cancel community subscription when subscription lifecycle ends. No params

Authencation

All endpoints related to authenication

Type Endpoint Value Params
POST /auth Login into account. Api with authenicate and authorize account and designate permissions based off of account role. String username, String password
POST /auth/logout Logs out and invalidates current session. No params
GET /auth/haspermissions Returns true if account has valid permissions. String permission
GET /auth/hasrole Returns true if account has valid role. String role

Retailer

All endpoints related to checkout and retail.

Type Endpoint Value Params
POST /retailer/checkout Checkout a customer and a list of items. Customer customer, JsonArray itemList

Model

All endpoints related to retreiving, updating, and deleting data.
Models currently include accountactivity, account, customer, item, and purchasesactivity. Replace the model in endpoints with one of the listed models.

Type Endpoint Value Params
GET model/all Get all model type's from database. No param
GET /model/all/whereparams Get all model type's with an array of conditions and a where clause. String whereClase, JsonArray params
GET /model/all/where Get all model types using only one condition and a where clause. String whereClase, String params
GET /model Get model by id Int id
DELETE /model Delete model by id Int id
POST /model Create a new model. Model model
PUT /model Update an existing model. Model model
